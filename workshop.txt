title=JaysServerLoot [BETA]
description=<-- BETA -->
description=
description=[b]Server-only mod.[/b] Clients do not need this installed.
description=
description=Adds configurable extra loot rolls to zombie corpses on top of vanilla loot. Never replaces or modifies existing drops — purely additive.
description=
description=Default config adds era-appropriate items for 1993 Kentucky: cigarette packs, loose cigarettes, disposable lighters, and matches at historically realistic drop rates. In 1993, roughly 25% of Kentucky adults smoked — zombies should reflect that.
description=
description=[b]Important:[/b] This mod does not replace vanilla zombie loot. It adds additional rolls on top of what PZ already gives. If you uninstall it, vanilla loot continues as normal.
description=
description=[h1]How It Works[/h1]
description=[list]
description=[*] When a zombie dies, the server rolls a configurable item table
description=[*] Won items are added directly to the corpse inventory
description=[*] Clients see items when they loot the body — no client mod needed
description=[*] All vanilla loot is untouched
description=[/list]
description=
description=[h1]Install[/h1]
description=[list]
description=[*] Subscribe on Workshop (server only)
description=[*] Add [b]JaysServerLoot[/b] to the [b]Mods=[/b] line in your server .ini
description=[*] Clients do NOT need to subscribe — this is server-side only
description=[*] Edit config.lua to customize items and chances (optional)
description=[/list]
description=
description=[h1]Configuration[/h1]
description=Edit the config file at:
description=[code]Contents/mods/JaysServerLoot/42/media/lua/server/JaysServerLoot/config.lua[/code]
description=
description=[b]Format:[/b] Items are defined as Module.ItemID:chance separated by semicolons.
description=[code]JaysServerLoot.ExtraRolls = 2
description=JaysServerLoot.Items = "Base.CigarettePack:0.25;Base.CigaretteSingle:0.15;Base.LighterDisposable:0.10;Base.Matches:0.08"[/code]
description=
description=[b]ExtraRolls[/b] — How many times the item table is rolled per zombie. Default: 2
description=[b]Chance[/b] — 0.0 to 1.0 per item (0.25 = 25% chance per roll)
description=

description=[h1]Validation[/h1]
description=On server start, every item ID is checked against PZ's item registry. Invalid items are logged and skipped — they won't crash the server.
description=[code][JaysServerLoot] Registered: Base.CigarettePack at 25.0% per roll
description=[JaysServerLoot] WARNING: Item 'Base.FakeItem' not found in game. Skipping.
description=[JaysServerLoot] Initialized with 4 item(s), 2 extra roll(s) per zombie[/code]
